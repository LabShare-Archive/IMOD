Slicer Angle Storage

This window allows you to store slicer angles and positions in a model and return to those positions in a slicer window. It has a table that shows all of the stored angles and positions. You can edit the values in the table as well as enter a label for each one. The angles can be obtained from the slicer only when you press a button, or a continuous update mode can be selected so that angles are recorded as they change. All stored angles are assigned a time value corresponding to the image file being displayed. When more than one image file is loaded into 3dmod, the table shows angles only for the current time. Each time has an independent value for the current row, so the current row may switch when you change times.

If there is more than one slicer window, the angle window always exchanges data with the last active slicer. A slicer window becomes the active window when you use a hot key, slider, or other control in that window.

Row operations. These buttons operate on one row of values in the table. A row can be made the current row by clicking in one of its cells, or by clicking the numbered button on the left. When you click one of those buttons, the slicer will be set to the angles in that row.

  • Get Angles will fetch the angles and position from the slicer into the current table row.
  • Set Angles will set the slicer to the angles and position in the current table row.
  • Delete will delete the current row of values.
  • New Row creates a new row in the table and stores the current slicer angles there.

Operations on volume angles sets. These controls appear when you have loaded more than one image file into 3dmod, or when the model file already has angles stored for more than one time value from a previous run with multiple image files. They operate on the entire set of angles shown in the table at once.

  • Remove deletes the angles shown in the table for the current time, and shifts the times of all angles at higher times down by one.
  • Insert effectively inserts an empty angle list for the current time by shifting the times of all angles at the current time and higher times up by one.
  • Renumber To moves the set of all angles at the current time to a different time, as selected in the spin box. Angles at times between the current time and the target time are shifted up or down by one, so the angle sets are just rearranged without losing any.
  • Copy From replaces all the entries at the current time with a copy of the angles from a different time, as selected in the spin box.

Continuous update. When this option is off, angles are fetched from a slicer only when Get Angles is pressed. The slicer is set to an angle when either Set Angles or one of the numbered buttons on the left are pressed. When the option is on, angle or position changes in a slicer automatically modify the stored values that are shown in the current row. When the time is changed, the slicer is automatically set to the current values for the new time ( which can be in a different row of the table). In addition, if you edit one of the cells, the values in that row are sent to the slicer.

Odds and Ends.

  • Undo and redo are available for all changes in stored angles.
  • The buttons on the left may not work 100% of the time as a substitute for Set Angles especially right after changing times.
  • To edit a cell, just click once and start typing to replace the current contents completely, or double click to edit the contents. Press "Esc" to abandon editing after clicking a cell. Press "Enter" when you are done editing a cell.
  • Labels can be up to 32 characters long. Cell columns can be resized with the mouse, which you might need to do to show a long label.
