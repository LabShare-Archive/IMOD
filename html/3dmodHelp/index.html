3dmod Help Index

3dmod is in transition to a new help system based on html files and Qt Assistant. If you set this window to your preferred size, it should come up at that size in the future. For older versions of Qt you may need to close the sidebar, and then it will not appear again until you open it.

The following topics are available so far:

3dmod image display windows:
Zap Window
Slicer Window

3dmod dialog boxes:
Contour joining
Contour moving
Image movie control
Image processing
Bead fixer

Model view dialog boxes:
Model view object edit
Model movie & montage control
Depth cueing
