3dmodv Menus

  • File Menu
    • Open Model: Load a new model to view.
    • Save Mode: Save the current model.
    • Save Model As...: Save model under a different name.
    • Snap TIFF As...: Save a snapshot to a specified TIFF file.
    • Snap NonT As...: Save a snapshot to a specified non-TIFF file.
    • Zero Snap File #: Reset the counter for snapshot files to 0.
    • Set Snap Dir...: Set a directory to put snapshots into. The same directory will be used for image snapshots.
    • Movie...: Program a sequence of displays and save them.
    • Close: Quit 3dmodv or close model view window in imod.
  • Edit Menu
    • Objects...: Open the object edit dialog.
    • Controls...: Open the display control dialog.
    • Object List...: Show objects by name with On/Off buttons.
    • Background...: Change the background color.
    • Models...: Control the display of multiple models.
    • Views...: Open a dialog to save and restore views.
    • Image...: Display an image slice on the model.
  • View Menu: Rendering options.
    • Double Buffer: Change between double and single buffer visual.
    • Lighting: Turn rendering with a light on or off.
    • Wireframe: Render data in wireframe only.
    • Low Res: Display low resolution mesh.
    • Stereo...: Open stereo control dialog.
    • Depth Cue...: Control dimming of display with distance.
