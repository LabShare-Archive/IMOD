Isosurface View Dialog

This dialog allows you to display isosurfaces of the input stack or a subvolume of it. An isosurface is a surface assuming a constant intensity value set by the user. The subvolume is specified by a bounding box.

The first three check boxes (View isosurfaces, etc.) turn on the display of the indicated objects.

Below the three check box is the histogram of voxels inside the bounding box. The Threshold slider sets the constant intensity value the isosurfaces will assume.

The X, Y, and Z sliders will track the current point coordinate in 3dmod image display windows and can be used to adjust the center position of the bounding box.

The X size, Y size, and Z size sliders can be used to set the X, Y, and Z size of the bounding box. The bounding box will center on the coordinate unless that would result in the box size being smaller than the slider value.
