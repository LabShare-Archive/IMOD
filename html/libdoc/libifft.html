IMOD Library libifft

The IMOD library libifft contains functions for performing 1-D, 2-D, and 3-D Fourier transforms, callable from C or Fortran. The FFT routines were written by Lynn Ten Eyck, wrapper routines were written by David Agard, and it was translated from Fortran to C by David Mastronarde.
